# ✨ Colorful QuoteSpark | Random Quote Generator

A beautiful, modern, and fully responsive random quote generator with smooth animations and backend integration.

## Features
- Beautiful glassmorphic + gradient design
- Real-time random quotes from **quotable.io API** (with local fallback)
- Share functionality (Web Share API + Clipboard fallback)
- Fully responsive (mobile-friendly)
- Keyboard shortcut (`N` key)
- Double-click card for new quote
- Loading animation & toast notifications

## How to Run
1. Open `index.html` in any modern browser
2. Click **"New Quote"** or press `N` to get fresh inspiration

## Tech Stack
- HTML5
- CSS3 (Flexbox, Gradients, Backdrop-filter)
- Vanilla JavaScript (Async/Await, Fetch API)
